Tutorial Membuat Website

Diposting pada 19 November 2023 oleh Aulia Imam Nugroho
Bahasa Indonesia Kompetensi Keahlian

Pertama

Saat membuat website, pertama kita membutuhkan Visual Studio Code sebagai tempat kerja coding yang akan kita lampirkan.

Kedua

Buatkan file baru dengan cara menekan New Text File pada bagian File di atas.

Ketiga

Untuk membuat web kita membutuhkan Style pada bagian webnya, disini sebagai contoh kami menggunakan Style yang tersedia di Google.

Keempat

Setelah menyelesaikan bagian Style, kita akan lanjut mengisi halaman utama web, disini kami beri nama home.html sebagai halaman utamanya.

Kelima

Selanjutnya kita perlu membuat Head dan Body pada website.

Keenam

Karena kami menggunakan bantuan Bootstrap jadi kami hanya perlu mengisi code dengan cara yang simple seperti digambar.

Ketujuh

Setelah halaman utama telah jadi dan sesuai keinginan, lanjut membuat halaman lainnya seperti Profile dan Contact sebagai contohnya.

Kedelapan

Pada halaman lainnya kalian bisa isi sesuai kemauan, disini kami mengisi sesuai judul yaitu Profile dan Contact, menggunakan class dari Bootstrap.

Kesembilan

Berikut adalah hasil websitenya.
